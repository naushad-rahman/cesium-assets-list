Name:
STK World Terrain
Description:
STK World Terrain is a high-resolution, worldwide terrain elevation tileset useful for Earth surface visualization and other uses. It continually grows in size and resolution as new sources of high-quality elevation data are found, acquired, and incorpoated into the tileset.

View in Cesium

Using this tileset in Cesium

// In an application that uses CesiumWidget:
var widget = new Cesium.CesiumWidget('cesiumContainer', {
    terrainProvider : new Cesium.CesiumTerrainProvider({
        url : '//assets.agi.com/stk-terrain/world'
    })
});

// In an application that uses Viewer:
var viewer = new Cesium.Viewer('cesiumContainer', {
    baseLayerPicker : false,
    terrainProvider : new Cesium.CesiumTerrainProvider({
        url : '//assets.agi.com/stk-terrain/world'
    })
});


// In other applications:
var scene = ...;
scene.terrainProvider = new Cesium.CesiumTerrainProvider({
    url : '//assets.agi.com/stk-terrain/world'
});
For a comprehensive overview on incorporating STK World Terrain into Cesium applications, refer to the Cesium Terrain and Water Tutorial

Offline and disconnected use

To host this tileset on your own infrastructure, you need STK Terrain Server. STK Terrain Server enables use of this tileset in an offline or disconnected environment, such as on a private network disconnected from the Internet. It also allows you to host the tileset closer to your users in order to improve performance, to overlay this tileset with your own terrain data, to create an entirely independent tileset from your own data, and to maintain full control over uptime and availability.
